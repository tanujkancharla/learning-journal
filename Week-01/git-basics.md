# Git Basics

## What is Git?

Git is a distributed version control system that tracks changes to files and allows developers to manage and collaborate on software projects.

## What is GitHub?

GitHub is a cloud platform that hosts Git repositories, making it easy to store code, collaborate with others, review changes, and manage software projects.

## What does `git add` do?

`git add` stages changes by moving modified or new files from the working directory into the staging area, preparing them for the next commit.

## What does `git commit` do?

`git commit` creates a snapshot of the staged changes in the local Git repository along with a descriptive commit message.

## What does `git push` do?

`git push` uploads your local commits to the remote repository on GitHub so they are backed up and available to collaborators.
